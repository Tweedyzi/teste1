Qu’est-ce que GameForgeAI ?

GameForgeAI est une plateforme qui vous permet de créer des jeux vidéo à partir d’un simple texte. Grâce à l’intelligence artificielle, vous pouvez générer des concepts, du code source, et même télécharger un projet jouable.

Fonctionnement

  • Vous décrivez votre jeu (ex : “RPG 2D avec quêtes et combats tour par tour”)
  • L’IA analyse votre prompt et propose un moteur (Godot, HTML5)
  • Un projet est généré automatiquement (scènes, code, assets)
  • Vous pouvez télécharger le projet complet en .zip

Technologies utilisées

  • 🎯 OpenAI GPT-4 pour la génération intelligente
  • 🎮 Godot Engine pour les jeux 2D/3D sur PC
  • 🌐 HTML5 + JS pour des jeux jouables dans un navigateur
  • 📦 PHP + JSON pour stocker vos projets
